_A starter blog built with Gatsby_

## Features

- Gatsby 4
- React 18
- TypeScript
- styled-components
- Prettier
- Path Mapping
- Syntax highlighting for your Markdown docs

## Quick start

To run this project locally:

```shell
npm i
gatsby develop
```
