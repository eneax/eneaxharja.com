Sort Object by property using JavaScript:

```js title="Console"
const topScorers = {
  Ronaldo: 770,
  Bican: 805,
  Pele: 767,
};

const playersSorted = Object.keys(topScorers).sort(
  (a, b) => topScorers[b] - topScorers[a]
);

console.log(playersSorted); // ["Bican", "Ronaldo", "Pele"]
```
