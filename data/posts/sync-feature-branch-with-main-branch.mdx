---
title: "Sync feature branch with main branch"
date: "2020-04-15"
description: "Keep your feature branch in sync with your main branch."
tags: ["git"]
---

If you are in the `feature-branch{:js}`, go the the `main{:js}` branch first:

```bash title="Terminal"
git checkout main
git pull
git checkout feature-branch
git pull --all
git merge main
```
