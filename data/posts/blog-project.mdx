## Features

- Gatsby 4
- React 18
- TypeScript
- styled-components
- Prettier
- Path Mapping
- Syntax highlighting for your Markdown docs

## Quick start

To run this project locally:

```shell title="Terminal"
npm i
gatsby develop
```
