## 🦄 Features

- ⚡️ Gatsby 4
- ⚛️ React 18
- ⛑ TypeScript
- 💅 styled-components
- 📏 ESLint
- 💖 Prettier
- 🗂 Path Mapping

## 🚀 Quick start

To run this project locally:

```shell
  npm i
  gatsby develop
```
