# eneaxharja [dot] com

Welcome to my own little corner of the Web!

## 🦄 Features

- Next.js 12
- React 18
- TypeScript 4
- Tailwind CSS 3
- Path Mapping (for absolute imports)

## 🚀 Quick start

To run this project locally:

```shell
npm i
npm run dev
```
